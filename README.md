# 🏡 Home Design Website

A modern home decoration website built with **React**, **TypeScript**, and **Vite**.
The platform provides creative interior design ideas and showcases ceramic products users can explore and purchase.

---

## ✨ Features

* Home decoration inspiration
* Modern and responsive UI
* Ceramic product showcase
* Fast performance with Vite
* Reusable React components
* Type-safe development with TypeScript

---

## 🛠️ Tech Stack

* React
* TypeScript
* Vite
* CSS / Tailwind CSS

---
